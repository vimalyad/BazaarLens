BazaarLens is an AI market intelligence PWA for small business owners built for iQOO Hackathon 2026 Bengaluru (13-14 Jun 2026). Prize pool INR 3.2L.

**Judging:** 25% iQOO Office Kit, 25% Phone-First, 20% AI-Native, 20% Real-World Utility, 10% Polish.

**Team strengths:** Next.js, FastAPI, AI/LLM, RAG — no native Android.

**Why:** Scoring "iQOO Office Kit" (25%) via Web Share API, camera-first UX, screen mirroring during demo — not via a developer SDK (none exists). The Office Kit is a cross-device productivity suite in OriginOS 6, not a developer API.

**How to apply:** Every architectural decision should tie back to the judging weights. Phone-first = ZXing camera, PWA push notifications. iQOO Office Kit = Web Share API + demo via screen mirror.
